
este proyecto contiene

---programas y versiones
- python 3.11.3
- jinja2 3.1.3
- flask 3.0.2 , flask-mysqldb 2.0.0
- mysql-connector-python 8.3.0 , mysqlclient 2.2.4
- python-dateutil 2.9.0


--- lo que hace
- inicio de sesion con bootstrap
tabla de usuario con 3 campos id,name y password
btn de mostrar password

-menu
botones a diferentes views

- layout
archivo de html que sirve de marco donde se meten otras vistas
con tiene css, js y navbar con menus y usuario
css mia ordenado

- view usuario
listar, agregar, eliminar y update usuarios
con formularios, modales, tablas

-view productos
listar, agregar, eliminar y update usuarios
con formularios con campo de cbo, modales tablas

-view empleados
listar, agregar, eliminar y update usuarios
con formularios con campos de radio button y datetime, modales, tablas
tabla con tipos enum para sexo, date fechas y longblob para imagenes
boton muestra y oculta td de imagenes tabla
